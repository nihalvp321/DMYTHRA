Dmythra – Empowering Differently Abled Individuals



🌟 Overview

Dmythra is a web platform designed to empower differently abled individuals by providing a space where they can showcase their talents, skills, and abilities while connecting with sponsors who are willing to support their growth and needs.

The platform creates a bridge between differently abled people and sponsors, enabling skill recognition, financial support, and community development.

🎯 Objectives
Provide a platform for differently abled individuals to showcase their skills.
Enable users to request assistance and support for their needs.
Allow sponsors to discover talented individuals and provide financial contributions.
Create transparency between users, sponsors, and administrators.
Promote inclusion, independence, and equal opportunities.
👥 User Roles
1. User (Differently Abled Person)

Users can:

Register and manage their profile.
Upload and showcase their skills and talents.
Add descriptions, achievements, and supporting information.
Request assistance for specific needs.
Track sponsorships and support received.
Communicate their goals and requirements.
2. Sponsor

Sponsors can:

Browse skill profiles uploaded by users.
View user achievements and talents.
Provide sponsorship or financial support.
Support specific skills, projects, or individual needs.
Track sponsorship history and contributions.
3. Administrator

Administrators can:

Manage users and sponsors.
Verify and moderate uploaded content.
Approve or reject skill submissions.
Monitor sponsorship activities.
Manage platform data and ensure system integrity.
Generate reports and platform statistics.

⚙️ Installation
Clone the Repository
git clone https://github.com/yourusername/dmythra.git
cd dmythra
Create Virtual Environment
python -m venv venv
Activate Virtual Environment
Windows
venv\Scripts\activate
Linux / macOS
source venv/bin/activate
Install Dependencies
pip install -r requirements.txt
Run the Application
python app.py

The application will be available at:

http://127.0.0.1:5000
