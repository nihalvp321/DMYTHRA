# 🌟 Dmythra - Empowering Differently Abled Individuals

Dmythra is a web-based platform designed to empower differently abled individuals by providing a space to showcase their skills, talents, and abilities while connecting them with sponsors who can support their growth and needs.

The platform serves as a bridge between differently abled people and sponsors, creating opportunities for recognition, financial assistance, and community support.

---

## 📖 Project Overview

Many differently abled individuals possess valuable skills and talents but often lack opportunities, visibility, and financial support. Dmythra addresses this challenge by providing a centralized platform where users can:

- Showcase their skills and talents.
- Request support for their personal or professional needs.
- Connect with sponsors willing to provide financial assistance.
- Gain recognition for their achievements and abilities.

Sponsors can explore skill profiles, identify talented individuals, and contribute directly to support their growth and development.

## 👥 User Roles

### 1. User (Differently Abled Individual)

Users can:

- Register and manage their profile.
- Upload and showcase their skills.
- Share achievements and talents.
- Submit requests for assistance or support.
- View sponsorships received.
- Track their support requests and contributions.

---

### 2. Sponsor

Sponsors can:

- Browse skill profiles uploaded by users.
- Discover talented individuals.
- Provide financial support for specific skills or needs.
- Track sponsorship history.
- View the impact of their contributions.

---

### 3. Administrator

Administrators can:

- Manage users and sponsors.
- Approve or moderate content.
- Monitor platform activities.
- Manage sponsorship records.
- Ensure data integrity and system security.
- Generate reports and platform statistics.

## 🚀 Installation Guide

### 1. Clone the Repository

```bash
git clone https://github.com/yourusername/dmythra.git
cd dmythra
```

### 2. Create a Virtual Environment

```bash
python -m venv venv
```

### 3. Activate the Virtual Environment

#### Windows

```bash
venv\Scripts\activate
```

#### Linux / macOS

```bash
source venv/bin/activate
```

### 4. Install Dependencies

```bash
pip install -r requirements.txt
```

### 5. Run the Application

```bash
python app.py
```

### 6. Open in Browser

```text
http://127.0.0.1:5000
```
